## 2. 기술 스택 선정 근거

### 2.1 전이금속: ECP 감지 + MINAO 폴백 전략

**문제의 핵심**: PySCF의 `lo.iao.iao()` 함수는 기본적으로 `minao='minao'` 참조 기저를 사용한다. 이 MINAO(Minimal Atomic Orbital) 기저는 H(Z=1)부터 Zn(Z=30)까지만 정의되어 있으나, ECP(Effective Core Potential) 기저(예: LANL2DZ)를 사용할 때 내부 전자가 제거된 상태에서 MINAO 기저와의 불일치가 발생한다.

**조사 결과**:

- PySCF `lo.iao()` 함수의 `minao` 파라미터를 통해 참조 기저를 변경할 수 있음 (`minao='sto-3g'` 등)
- ORCA 6.1은 atomic SCF MO를 참조로 사용하여 이 문제를 우회하지만, PySCF에는 해당 구현이 없음
- `mol.has_ecp()` 메서드로 ECP 사용 여부를 런타임에 감지 가능
- def2-SVP는 all-electron 기저이므로 3d 전이금속에서도 MINAO가 정상 동작함

**채택 전략**: ECP 감지 시 자동으로 `minao='sto-3g'`로 폴백하되, 사용자가 명시적으로 지정한 값은 그대로 사용. 경고 메시지를 반환하여 "def2-SVP 같은 all-electron 기저를 권장" 안내.

**검증 분자 선정 근거**:

| 분자                   | 화학식  | 왜 이 분자인가                                                                                                  | 기저     | 예상 IBO 범위 |
| ---------------------- | ------- | --------------------------------------------------------------------------------------------------------------- | -------- | ------------- |
| Iron pentacarbonyl     | Fe(CO)₅ | 배위 화학 교과서의 대표 예제. 18전자 규칙, CO σ-donation + π-backdonation 해석이 풍부. Knizia 원논문에서도 인용 | def2-SVP | 20~35         |
| Titanium tetrachloride | TiCl₄   | 초기 전이금속(d⁰). 순수 σ-결합만 존재하여 IBO 해석이 명확. d 전자 없음 → IBO 해석의 baseline                    | def2-SVP | 18~30         |

IBO 개수를 범위로 지정하는 이유: 전이금속의 d 오비탈 참여 정도에 따라 로컬라이제이션 결과가 달라질 수 있고, 좌표가 이상적 구조(실험 구조가 아닌 대략적 대칭 구조)이므로 정확한 개수보다 "IBO가 합리적 범위 내에서 생성됨"을 검증하는 것이 적절하다.

### 2.2 UHF/ROHF: Alpha/Beta 분리 로컬라이제이션

**문제의 핵심**: RHF(Restricted Hartree-Fock)에서는 `mf.mo_coeff`가 단일 ndarray이고 `mf.mo_occ`도 단일 ndarray이다. UHF(Unrestricted HF)에서는 이들이 `(ndarray_alpha, ndarray_beta)` **튜플**로 반환된다. 현재 코드의 `compute_iao()`와 `compute_ibo()`는 이 차이를 처리하지 못한다.

**조사 결과**:

- PySCF의 `lo.iao.iao()` 및 `lo.ibo.ibo()` 함수 자체는 "단일 스핀 채널의 점유 MO 계수"를 받으므로, alpha와 beta를 **각각 별도로** 호출하면 정상 동작
- PySCF GitHub Issue #2398 (2024-09): UHF localized natural orbitals 논의에서 이 접근법이 확인됨
- `fast_iao_mullikan_pop`은 UHF dm(리스트)을 이미 지원하여 Mulliken 전하는 직접 계산 가능
- ROHF(Restricted Open-Shell HF)는 alpha/beta 공간 오비탈이 동일하므로 UHF보다 단순

**채택 전략**:

1. 새 메서드 `compute_scf_flexible(spin=N)`으로 spin>0일 때 자동 UHF 선택
2. `compute_iao_uhf()`, `compute_ibo_uhf()`로 alpha/beta 각각 처리
3. 결과를 `{"alpha": {...}, "beta": {...}, "total_ibo": N}` 형태로 구조화
4. 기존 RHF 경로는 **절대 수정하지 않음** → RHF 코드가 `isinstance` 체크 없이 그대로 동작

**검증 분자**:

| 분자        | 스핀 다중도    | 검증 포인트                                |
| ----------- | -------------- | ------------------------------------------ |
| O₂ (삼중항) | spin=2, 삼중항 | alpha IBO > beta IBO (2개 unpaired e⁻)     |
| NO (라디칼) | spin=1, 이중항 | alpha IBO = beta IBO + 1 (1개 unpaired e⁻) |

### 2.3 헤드리스 PNG: Playwright + SwiftShader

**문제의 핵심**: py3Dmol은 HTML 기반 인터랙티브 시각화를 생성하지만, 논문에 삽입할 정적 PNG 이미지를 직접 생성하는 기능이 없다. 연구자는 브라우저에서 스크린샷을 수동으로 캡처해야 하며, 이는 자동화와 재현성에 반한다.

**기술 비교**:

| 도구             | 언어    | WebGL 지원         | 속도 | 헤드리스 안정성                      |
| ---------------- | ------- | ------------------ | ---- | ------------------------------------ |
| **Playwright**   | Python  | SwiftShader 내장   | 빠름 | 우수 (Chromium 네이티브)             |
| Puppeteer        | Node.js | 동일               | 빠름 | 우수 (Python 프로젝트와 별개 런타임) |
| Selenium         | Python  | 드라이버 설정 필요 | 느림 | 취약                                 |
| kaleido (plotly) | Python  | 불가 (SVG 전용)    | —    | 3Dmol.js 미지원                      |

**Playwright 채택 이유**:

1. **Python 네이티브**: `pip install playwright && playwright install chromium`으로 설치 완료. Node.js 런타임 불필요
2. **SwiftShader 내장**: Chromium에 포함된 소프트웨어 OpenGL ES 구현. GPU 없는 CI/CD 서버나 WSL2에서도 WebGL 렌더링 가능
3. **검증된 사례**: Promaton 블로그(2025-02)에서 3D 웹앱의 Playwright 테스트 성공 사례 보고. Playwright Issue #11627에서 headless WebGL 지원 확인
4. **핵심 Chromium 인자**: `--enable-gpu --use-angle=swiftshader-webgl --enable-unsafe-swiftshader`로 소프트웨어 렌더링 강제

**해상도 전략**: `device_scale_factor=2`로 Retina 품질(1600×1200 실효 해상도)의 PNG 생성. 논문 Figure 요구사항(300 DPI, 약 5×4 inch = 1500×1200 px)을 충족.

**WSL2 주의사항**: headless=True 필수, GPU passthrough 불가, SwiftShader 폴백 필수. 추가 시스템 의존성: `libnss3 libatk-bridge2.0-0 libxkbcommon0 libgbm1`.

## 3. Phase ζ-1: 전이금속 IAO/IBO 지원

### 3.1 코드 변경: `pyscf_backend.py`

**변경 1**: `_resolve_minao()` 헬퍼 메서드 추가

```python
def _resolve_minao(self, mol, minao='minao'):
    """ECP 감지 시 minao 자동 폴백.

    Args:
        mol: PySCF Mole 객체.
        minao: 사용자 지정 참조 기저. 'minao'(기본값)이면 ECP 시 자동 폴백.

    Returns:
        tuple: (effective_minao: str, warnings: list[str])

    동작:
        1. mol.has_ecp() 호출
        2. ECP 존재 + minao=='minao' → 'sto-3g'로 전환 + 경고
        3. 사용자가 minao를 명시적으로 지정했으면 ('minao' 외) 그대로 사용
    """
    warnings = []
    effective = minao
    if hasattr(mol, 'has_ecp') and mol.has_ecp() and minao == 'minao':
        effective = 'sto-3g'
        warnings.append(
            "ECP detected. Switched IAO reference basis from 'minao' to 'sto-3g'. "
            "For best results with transition metals, use all-electron "
            "basis (e.g., def2-SVP) without ECP."
        )
    return effective, warnings
```

이 메서드가 별도로 분리된 이유: (1) ECP 감지 로직을 단위 테스트 가능하게 만듦, (2) UHF 경로(`compute_iao_uhf`)에서도 동일 로직 재사용, (3) 향후 ORCA 기반 atomic SCF MO 참조가 구현되면 이 메서드만 확장.

**변경 2**: `compute_iao()` 시그니처에 `minao` 파라미터 추가

```python
def compute_iao(self, scf_result, mol, minao="minao"):
    # ...
    effective, warnings = self._resolve_minao(mol, minao)
    iao_coeff = lo.iao.iao(mol, mo_occ, minao=effective)
    # ...
```

기존 호출은 `compute_iao(scf_result, mol)`로 `minao` 없이 호출하므로 하위 호환성이 완전히 유지된다.

```python
"""Phase ζ-1: 전이금속 IAO/IBO 테스트."""
import pytest
pyscf = pytest.importorskip("pyscf")

from qcviz_mcp.backends.pyscf_backend import PySCFBackend

backend = PySCFBackend()
WATER = "O 0 0 0; H 0 0.757 0.587; H 0 -0.757 0.587"


class TestTransitionMetal:
    def test_ecp_detection_lanl2dz(self):
        """ECP 기저(LANL2DZ) 감지."""
        from pyscf import gto
        mol = gto.M(atom="Fe 0 0 0", basis="lanl2dz", spin=4)
        assert mol.has_ecp()

    def test_allelectron_no_ecp(self):
        """All-electron 기저(def2-SVP)는 ECP 없음."""
        from pyscf import gto
        mol = gto.M(atom="Fe 0 0 0", basis="def2-svp", spin=4)
        assert not mol.has_ecp()

    def test_resolve_minao_ecp(self):
        """ECP 시 minao 자동 폴백 → 'sto-3g'."""
        from pyscf import gto
        mol = gto.M(atom="Fe 0 0 0", basis="lanl2dz", spin=4)
        effective, warnings = backend._resolve_minao(mol, "minao")
        assert effective == "sto-3g"
        assert len(warnings) > 0
        assert "ECP" in warnings[0]

    def test_resolve_minao_no_ecp(self):
        """ECP 없으면 minao 그대로 유지."""
        from pyscf import gto
        mol = gto.M(atom=WATER, basis="sto-3g")
        effective, warnings = backend._resolve_minao(mol, "minao")
        assert effective == "minao"
        assert len(warnings) == 0

    def test_minao_explicit_override(self):
        """사용자 명시적 'sto-3g' → ECP 여부 무관하게 그대로."""
        from pyscf import gto
        mol = gto.M(atom=WATER, basis="sto-3g")
        effective, warnings = backend._resolve_minao(mol, "sto-3g")
        assert effective == "sto-3g"

    def test_water_ibo_with_minao_param(self):
        """water + minao='sto-3g' 명시적 전달 — IBO 5개."""
        scf_res, mol = backend.compute_scf(WATER, basis="sto-3g")
        iao_res = backend.compute_iao(scf_res, mol, minao="sto-3g")
        ibo_res = backend.compute_ibo(scf_res, iao_res, mol)
        assert ibo_res.coefficients.shape[1] == 5
```

**완료 조건**: 6개 테스트 전부 통과. Fe(CO)₅ 전체 파이프라인 테스트는 `@pytest.mark.slow`로 마크하여 일상적 CI에서 제외하고, 릴리스 전 수동 실행.

## 4. Phase ζ-2: UHF/ROHF IBO 완전 지원

### 4.1 코드 변경: `pyscf_backend.py`

**핵심 설계 결정**: 기존 `compute_scf()`, `compute_iao()`, `compute_ibo()` 메서드를 수정하지 않고, 새 메서드(`compute_scf_flexible()`, `compute_iao_uhf()`, `compute_ibo_uhf()`)를 **병렬로 추가**한다.

이유:

1. 기존 95개 테스트가 RHF 경로에 의존 → 수정 시 regression 위험
2. UHF 로직은 alpha/beta 분리가 본질적으로 다른 데이터 구조를 요구
3. MCP 도구 레벨에서 `isinstance(mf, UHF)` 체크로 분기하는 것이 명확

**변경 1**: `compute_scf_flexible()` — spin 자동 감지

```python
def compute_scf_flexible(self, atom_spec, basis="sto-3g", charge=0, spin=0):
    """spin>0 시 UHF 자동 선택.

    UHF 반환 시 mf.mo_coeff는 (ndarray_alpha, ndarray_beta) 튜플.
    RHF 반환 시 mf.mo_coeff는 단일 ndarray.
    """
    from pyscf import gto, scf
    atom_spec = _parse_atom_spec(atom_spec)
    mol = gto.M(atom=atom_spec, basis=basis, charge=charge, spin=spin, verbose=0)
    if spin > 0:
        mf = scf.UHF(mol)
    else:
        mf = scf.RHF(mol)
    mf.kernel()
    if not mf.converged:
        raise RuntimeError(f"SCF not converged for spin={spin}")
    return mf, mol
```

**변경 2**: `compute_iao_uhf()` — alpha/beta 분리 IAO

```python
def compute_iao_uhf(self, mf, mol, minao="minao"):
    """UHF 전용 IAO: alpha/beta 스핀 채널을 각각 처리.

    Returns:
        dict: {
            'alpha': {'iao_coeff': ndarray, 'n_iao': int},
            'beta':  {'iao_coeff': ndarray, 'n_iao': int},
            'is_uhf': True,
            'minao_used': str,
            'warnings': list[str]
        }
    """
    from pyscf import lo
    effective, warnings = self._resolve_minao(mol, minao)

    mo_a, mo_b = mf.mo_coeff      # UHF: 튜플
    occ_a, occ_b = mf.mo_occ      # UHF: 튜플

    mo_occ_a = mo_a[:, occ_a > 0]  # alpha 점유 MO
    mo_occ_b = mo_b[:, occ_b > 0]  # beta 점유 MO

    iao_a = lo.iao.iao(mol, mo_occ_a, minao=effective)
    iao_b = lo.iao.iao(mol, mo_occ_b, minao=effective)

    return {
        "alpha": {"iao_coeff": iao_a, "n_iao": iao_a.shape[1]},
        "beta":  {"iao_coeff": iao_b, "n_iao": iao_b.shape[1]},
        "is_uhf": True,
        "minao_used": effective,
        "warnings": warnings,
    }
```

**변경 3**: `compute_ibo_uhf()` — alpha/beta 분리 IBO

```python
def compute_ibo_uhf(self, mf, iao_result, mol):
    """UHF 전용 IBO: alpha/beta 스핀 채널을 각각 로컬라이즈.

    Alpha에 unpaired 전자가 있으므로 alpha_n_ibo > beta_n_ibo가 일반적.
    """
    from pyscf import lo

    mo_a, mo_b = mf.mo_coeff
    occ_a, occ_b = mf.mo_occ
    mo_occ_a = mo_a[:, occ_a > 0]
    mo_occ_b = mo_b[:, occ_b > 0]

    ibo_a = lo.ibo.ibo(mol, mo_occ_a, iaos=iao_result["alpha"]["iao_coeff"])
    ibo_b = lo.ibo.ibo(mol, mo_occ_b, iaos=iao_result["beta"]["iao_coeff"])

    return {
        "alpha": {"ibo_coeff": ibo_a, "n_ibo": ibo_a.shape[1]},
        "beta":  {"ibo_coeff": ibo_b, "n_ibo": ibo_b.shape[1]},
        "is_uhf": True,
        "total_ibo": ibo_a.shape[1] + ibo_b.shape[1],
    }
```

**변경 4**: `compute_uhf_charges()` — UHF Mulliken 전하

```python
def compute_uhf_charges(self, mf, mol):
    """UHF Mulliken 전하 계산.

    UHF 밀도 행렬은 (dm_alpha, dm_beta) 튜플.
    총 밀도 = dm_alpha + dm_beta → 표준 Mulliken population.
    """
    import numpy as np
    dm = mf.make_rdm1()  # (dm_a, dm_b) for UHF
    s = mol.intor("int1e_ovlp")
    dm_total = dm[0] + dm[1] if isinstance(dm, (list, tuple)) else dm

    pop = np.einsum("ij,ji->i", dm_total, s)
    charges = []
    offset = 0
    for ia in range(mol.natm):
        nao_atom = sum(
            (2 * mol.bas_angular(ib) + 1) * mol.bas_nctr(ib)
            for ib in range(mol.nbas)
            if mol.bas_atom(ib) == ia
        )
        q = mol.atom_charge(ia) - sum(pop[offset:offset + nao_atom])
        charges.append(float(q))
        offset += nao_atom
    return charges
```

```python
"""Phase ζ-2: UHF/ROHF IBO 테스트."""
import pytest
pyscf = pytest.importorskip("pyscf")

from qcviz_mcp.backends.pyscf_backend import PySCFBackend

backend = PySCFBackend()


class TestUHFIBO:
    def test_o2_triplet_scf_converges(self):
        """O₂ 삼중항 UHF SCF 수렴."""
        mf, mol = backend.compute_scf_flexible(
            "O 0 0 0; O 0 0 1.208", "sto-3g", spin=2
        )
        assert mf.converged
        assert isinstance(mf.mo_coeff, (tuple, list))

    def test_o2_triplet_iao_uhf(self):
        """O₂ 삼중항 IAO: alpha/beta 분리."""
        mf, mol = backend.compute_scf_flexible(
            "O 0 0 0; O 0 0 1.208", "sto-3g", spin=2
        )
        iao_res = backend.compute_iao_uhf(mf, mol)
        assert iao_res["is_uhf"]
        assert iao_res["alpha"]["n_iao"] > 0
        assert iao_res["beta"]["n_iao"] > 0

    def test_o2_triplet_ibo_alpha_gt_beta(self):
        """O₂ 삼중항 IBO: alpha > beta (2 unpaired e⁻)."""
        mf, mol = backend.compute_scf_flexible(
            "O 0 0 0; O 0 0 1.208", "sto-3g", spin=2
        )
        iao_res = backend.compute_iao_uhf(mf, mol)
        ibo_res = backend.compute_ibo_uhf(mf, iao_res, mol)
        assert ibo_res["alpha"]["n_ibo"] > ibo_res["beta"]["n_ibo"]
        assert ibo_res["total_ibo"] > 0

    def test_no_radical_ibo(self):
        """NO 라디칼 (doublet): alpha = beta + 1."""
        mf, mol = backend.compute_scf_flexible(
            "N 0 0 0; O 0 0 1.151", "sto-3g", spin=1
        )
        iao_res = backend.compute_iao_uhf(mf, mol)
        ibo_res = backend.compute_ibo_uhf(mf, iao_res, mol)
        assert ibo_res["alpha"]["n_ibo"] == ibo_res["beta"]["n_ibo"] + 1

    def test_uhf_charges_sum_zero(self):
        """UHF Mulliken 전하 합 ≈ 0 (중성 분자)."""
        mf, mol = backend.compute_scf_flexible(
            "O 0 0 0; O 0 0 1.208", "sto-3g", spin=2
        )
        charges = backend.compute_uhf_charges(mf, mol)
        assert abs(sum(charges)) < 0.01
        assert len(charges) == 2  # O₂는 2원자

    def test_rhf_path_unchanged(self):
        """RHF (spin=0)는 mo_coeff가 단일 ndarray."""
        mf, mol = backend.compute_scf_flexible(
            "O 0 0 0; H 0 0.757 0.587; H 0 -0.757 0.587", "sto-3g", spin=0
        )
        assert not isinstance(mf.mo_coeff, (tuple, list))
        assert mf.converged
```

### 5.2 핵심 코드: `png_exporter.py`

```python
"""
헤드리스 PNG 내보내기.
Playwright + Chromium + SwiftShader로 py3Dmol HTML → PNG 캡처.

선택적 의존성: pip install playwright && playwright install chromium
WSL2:       sudo apt install libnss3 libatk-bridge2.0-0 libxkbcommon0 libgbm1
"""
import asyncio
from pathlib import Path
from typing import Optional


async def html_to_png(
    html_path: str,
    png_path: Optional[str] = None,
    width: int = 800,
    height: int = 600,
    wait_ms: int = 3000,
    timeout_ms: int = 30000,
) -> dict:
    """HTML 파일을 PNG로 캡처.

    Args:
        html_path: py3Dmol 생성 HTML 파일 경로.
        png_path:  출력 PNG 경로. None이면 입력 파일 확장자를 .png로 변경.
        width:     뷰포트 너비 (px). 실제 PNG는 device_scale_factor=2로 2배.
        height:    뷰포트 높이.
        wait_ms:   3Dmol.js 렌더링 완료 대기 시간 (ms).
        timeout_ms: 페이지 로딩 전체 타임아웃 (ms).

    Returns:
        dict: {success: bool, png_path: str, width: int, height: int,
               file_size_bytes: int, error: str (실패 시)}

    Note:
        3Dmol.js는 WebGL(canvas)을 사용하여 렌더링하므로,
        WebGL이 초기화된 후 스크린샷을 캡처해야 한다.
        wait_ms가 부족하면 빈 캔버스가 캡처될 수 있다.
    """
    try:
        from playwright.async_api import async_playwright
    except ImportError:
        return {
            "success": False,
            "error": (
                "Playwright not installed. "
                "Run: pip install playwright && playwright install chromium"
            ),
        }

    if png_path is None:
        png_path = str(Path(html_path).with_suffix(".png"))

    html_uri = Path(html_path).resolve().as_uri()

    try:
        async with async_playwright() as p:
            browser = await p.chromium.launch(
                headless=True,
                args=[
                    "--enable-gpu",
                    "--use-angle=swiftshader-webgl",
                    "--enable-unsafe-swiftshader",
                    "--no-sandbox",
                ],
            )
            context = await browser.new_context(
                viewport={"width": width, "height": height},
                device_scale_factor=2,  # Retina 품질
            )
            page = await context.new_page()
            await page.goto(html_uri, timeout=timeout_ms)

            # 3Dmol.js 렌더링 완료 대기
            await page.wait_for_timeout(wait_ms)

            # WebGL 캔버스 존재 및 크기 확인
            canvas_ok = await page.evaluate(
                "() => { const c = document.querySelector('canvas'); "
                "return c !== null && c.width > 0; }"
            )
            if not canvas_ok:
                # 첫 대기로 부족했으면 추가 대기
                await page.wait_for_timeout(wait_ms * 2)

            await page.screenshot(path=png_path, type="png")
            await browser.close()

        file_size = Path(png_path).stat().st_size
        return {
            "success": True,
            "png_path": png_path,
            "width": width * 2,
            "height": height * 2,
            "file_size_bytes": file_size,
        }
    except Exception as e:
        return {"success": False, "error": str(e)}


def html_to_png_sync(html_path: str, **kwargs) -> dict:
    """html_to_png의 동기 래퍼. 이벤트 루프가 없는 환경에서 사용."""
    return asyncio.run(html_to_png(html_path, **kwargs))
```
