## 전수조사 결과 (기술 스택 선정 근거)

### 전이금속 (문제1)

- PySCF `lo.iao(mol, orbocc, minao='minao')` — `minao` 파라미터로 참조 기저 변경 가능
- ECP 사용 시 기본 `'minao'` 기저가 부적합 → `'sto-3g'` 또는 all-electron 기저 사용 필요
- PySCF MINAO 기저는 H(Z=1)~Zn(Z=30)까지 3d 전이금속을 포함
- ORCA 6.1은 atomic SCF MO를 참조로 사용 (더 견고하지만 PySCF에는 미구현)
- **채택 전략**: ECP 감지 시 자동으로 `minao='sto-3g'`로 폴백 + all-electron 기저 권장
- **검증 분자**: [Fe(CO)₅] (iron pentacarbonyl, 18e, 오비탈 해석 풍부한 교과서 예제)

### UHF/ROHF (문제2)

- PySCF UHF `mf.mo_coeff` = `(ndarray_alpha, ndarray_beta)` 튜플
- `mf.mo_occ` = `(occ_alpha, occ_beta)` 튜플
- `lo.iao()`와 `lo.ibo()`는 alpha/beta를 **각각 별도로** 처리해야 함
- `fast_iao_mullikan_pop`은 UHF dm(리스트)을 이미 지원
- PySCF issue #2398: UHF localized natural orbitals 논의 (2024-09)
- **채택 전략**: alpha/beta 오비탈을 분리 로컬라이즈, 결과를 `{alpha: {...}, beta: {...}}` 형태로 반환
- **검증 분자**: O₂ 삼중항 (기존 δ-4에서 정의), NO 라디칼 (doublet)

### 헤드리스 PNG (문제3)

- Playwright Python이 최적 (Puppeteer는 Node.js, Selenium은 느림)
- 3Dmol.js(py3Dmol 기반)는 WebGL 사용 → headless에서 빈 캔버스 문제 발생 가능
- **해결**: Chromium 인자 `--enable-gpu --use-angle=swiftshader-webgl` (GPU 없는 서버에서도 작동)
- Playwright issue #11627: headless WebGL 지원 확인 (2022)
- Promaton 블로그 (2025-02): 3D 앱 Playwright 테스트 성공 사례
- **채택 전략**: Playwright 선택적 의존성 (`pip install playwright`), 빌트인 SwiftShader로 소프트웨어 렌더링
- **주의**: WSL2에서는 headless=True 필수, GPU 가속 불가 → SwiftShader 폴백

def compute_iao(self, scf_result, mol, minao='minao'):
    """
    IAO 계산. ECP 감지 시 자동 폴백.

    변경점:
    - mol.has_ecp() 검사 추가
    - ECP 감지 시: minao='sto-3g'로 자동 전환 + 경고 메시지
    - 사용자가 명시적으로 minao를 지정하면 그대로 사용
    """
    from pyscf import lo

    # ECP 자동 감지 및 폴백
    effective_minao = minao
    warnings = []
    if mol.has_ecp() and minao == 'minao':
        effective_minao = 'sto-3g'
        warnings.append(
            "ECP detected. Switched IAO reference basis from 'minao' to 'sto-3g'. "
            "For best results with transition metals, consider using all-electron "
            "basis sets (e.g., def2-SVP) without ECP."
        )

    mo_occ = scf_result.mo_coeff[:, scf_result.mo_occ > 0]
    iao_coeff = lo.iao.iao(mol, mo_occ, minao=effective_minao)

    return IAOResult(coefficients=iao_coeff, warnings=warnings, minao_used=effective_minao)

```python
def compute_scf(self, atom_spec, basis="sto-3g", charge=0, spin=0, method="auto"):
    """
    변경점: method="auto" 시 spin>0이면 UHF 자동 선택.
    """
    from pyscf import gto, scf
    mol = gto.M(atom=atom_spec, basis=basis, charge=charge, spin=spin)

    if method == "auto":
        if spin > 0:
            mf = scf.UHF(mol)
        else:
            mf = scf.RHF(mol)
    elif method == "uhf":
        mf = scf.UHF(mol)
    elif method == "rohf":
        mf = scf.ROHF(mol)
    else:
        mf = scf.RHF(mol)

    mf.kernel()
    return mf, mol

def compute_iao_uhf(self, mf, mol, minao='minao'):
    """
    UHF 전용 IAO 계산. alpha/beta를 분리 처리.

    Returns:
        dict: {
            'alpha': {'iao_coeff': ndarray, 'n_iao': int},
            'beta':  {'iao_coeff': ndarray, 'n_iao': int},
            'is_uhf': True
        }
    """
    from pyscf import lo

    mo_a, mo_b = mf.mo_coeff
    occ_a, occ_b = mf.mo_occ

    mo_occ_a = mo_a[:, occ_a > 0]
    mo_occ_b = mo_b[:, occ_b > 0]

    effective_minao = self._resolve_minao(mol, minao)  # ECP 폴백 로직 재사용

    iao_a = lo.iao.iao(mol, mo_occ_a, minao=effective_minao)
    iao_b = lo.iao.iao(mol, mo_occ_b, minao=effective_minao)

    return {
        'alpha': {'iao_coeff': iao_a, 'n_iao': iao_a.shape[1]},
        'beta':  {'iao_coeff': iao_b, 'n_iao': iao_b.shape[1]},
        'is_uhf': True,
        'minao_used': effective_minao
    }

def compute_ibo_uhf(self, mf, iao_result, mol):
    """
    UHF 전용 IBO 계산. alpha/beta 각각 로컬라이즈.
    """
    from pyscf import lo

    mo_a, mo_b = mf.mo_coeff
    occ_a, occ_b = mf.mo_occ
    mo_occ_a = mo_a[:, occ_a > 0]
    mo_occ_b = mo_b[:, occ_b > 0]

    ibo_a = lo.ibo.ibo(mol, mo_occ_a, iaos=iao_result['alpha']['iao_coeff'])
    ibo_b = lo.ibo.ibo(mol, mo_occ_b, iaos=iao_result['beta']['iao_coeff'])

    return {
        'alpha': {'ibo_coeff': ibo_a, 'n_ibo': ibo_a.shape[1]},
        'beta':  {'ibo_coeff': ibo_b, 'n_ibo': ibo_b.shape[1]},
        'is_uhf': True,
        'total_ibo': ibo_a.shape[1] + ibo_b.shape[1]
    }
```

## Phase ζ-3: 헤드리스 PNG 내보내기

### 목표

py3Dmol HTML 시각화를 Playwright를 사용해 자동으로 PNG 이미지로 캡처한다.

### 기술 스택

- **Playwright** (Python): `pip install playwright && playwright install chromium`
- **SwiftShader**: Chromium 내장 소프트웨어 WebGL 렌더러 (GPU 불필요)
- **핵심 인자**: `--enable-gpu --use-angle=swiftshader-webgl --enable-unsafe-swiftshader`

### 작업 내용

1. **새 모듈** — `src/qcviz_mcp/renderers/png_exporter.py`:

```python
"""
헤드리스 PNG 내보내기 모듈.
Playwright + Chromium + SwiftShader를 사용하여
py3Dmol HTML을 정적 PNG로 캡처.
"""
import asyncio
from pathlib import Path
from typing import Optional

async def html_to_png(
    html_path: str,
    png_path: Optional[str] = None,
    width: int = 800,
    height: int = 600,
    wait_ms: int = 3000,  # 3Dmol.js 렌더링 대기
    timeout_ms: int = 30000
) -> dict:
    """
    HTML 파일을 PNG로 캡처.

    Args:
        html_path: py3Dmol 생성 HTML 파일 경로
        png_path: 출력 PNG 경로 (None이면 자동 생성)
        width: 뷰포트 너비 (px)
        height: 뷰포트 높이 (px)
        wait_ms: 렌더링 완료 대기 시간 (ms)
        timeout_ms: 전체 타임아웃 (ms)

    Returns:
        dict: png_path, width, height, file_size_bytes, success
    """
    try:
        from playwright.async_api import async_playwright
    except ImportError:
        return {
            "success": False,
            "error": "Playwright not installed. Run: pip install playwright && playwright install chromium"
        }

    if png_path is None:
        png_path = str(Path(html_path).with_suffix('.png'))

    html_uri = Path(html_path).resolve().as_uri()

    async with async_playwright() as p:
        browser = await p.chromium.launch(
            headless=True,
            args=[
                '--enable-gpu',
                '--use-angle=swiftshader-webgl',
                '--enable-unsafe-swiftshader',
                '--disable-software-rasterizer',
                '--no-sandbox',
            ]
        )

        context = await browser.new_context(
            viewport={'width': width, 'height': height},
            device_scale_factor=2  # Retina 품질
        )

        page = await context.new_page()
        await page.goto(html_uri, timeout=timeout_ms)

        # 3Dmol.js 렌더링 완료 대기
        await page.wait_for_timeout(wait_ms)

        # WebGL 캔버스가 렌더링되었는지 확인
        canvas_exists = await page.evaluate('''
            () => {
                const canvas = document.querySelector('canvas');
                return canvas !== null && canvas.width > 0;
            }
        ''')

        if not canvas_exists:
            # 폴백: 더 오래 대기
            await page.wait_for_timeout(wait_ms * 2)

        await page.screenshot(path=png_path, type='png')
        await browser.close()

    file_size = Path(png_path).stat().st_size

    return {
        "success": True,
        "png_path": png_path,
        "width": width * 2,   # device_scale_factor=2
        "height": height * 2,
        "file_size_bytes": file_size
    }


def html_to_png_sync(html_path: str, **kwargs) -> dict:
    """동기 래퍼."""
    return asyncio.run(html_to_png(html_path, **kwargs))
```

2. **MCP 도구 확장** — `visualize_orbital`에 PNG 옵션 추가:

```python
async def visualize_orbital(
    cube_path: str,
    output_format: str = "html",  # "html" | "png" | "both"
    width: int = 800,
    height: int = 600
) -> dict:
    """
    output_format="png" 시 HTML 생성 후 자동 PNG 캡처.
    """
    # 1. 기존 HTML 생성 경로
    html_result = py3dmol_backend.render(cube_path)

    if output_format in ("png", "both"):
        from qcviz_mcp.renderers.png_exporter import html_to_png
        png_result = await html_to_png(html_result["html_path"])
        html_result["png_path"] = png_result.get("png_path")
        html_result["png_size"] = png_result.get("file_size_bytes")

    return html_result
```

```python
import pytest
import os

# Playwright 설치 여부에 따라 skip
playwright_available = False
try:
    import playwright
    playwright_available = True
except ImportError:
    pass

@pytest.mark.skipif(not playwright_available, reason="Playwright not installed")
class TestPNGExport:

    @pytest.fixture
    def sample_html(self, tmp_path):
        """간단한 3Dmol.js HTML 생성."""
        html = '''<!DOCTYPE html>
        <html><head>
        <script src="https://3Dmol.org/build/3Dmol-min.js"></script>
        </head><body>
        <div id="viewer" style="width:400px;height:300px;"></div>
        <script>
        var viewer = $3Dmol.createViewer("viewer", {backgroundColor:"white"});
        viewer.addSphere({center:{x:0,y:0,z:0}, radius:1.0, color:"red"});
        viewer.zoomTo();
        viewer.render();
        </script>
        </body></html>'''
        path = tmp_path / "test_mol.html"
        path.write_text(html)
        return str(path)

    @pytest.mark.asyncio
    async def test_html_to_png_basic(self, sample_html, tmp_path):
        """기본 PNG 캡처."""
        from qcviz_mcp.renderers.png_exporter import html_to_png
        png_path = str(tmp_path / "output.png")
        result = await html_to_png(sample_html, png_path=png_path)
        assert result["success"]
        assert os.path.exists(png_path)
        assert result["file_size_bytes"] > 1000  # 빈 이미지가 아님

    @pytest.mark.asyncio
    async def test_png_dimensions(self, sample_html, tmp_path):
        """지정된 크기로 캡처."""
        from qcviz_mcp.renderers.png_exporter import html_to_png
        result = await html_to_png(sample_html, width=1024, height=768)
        assert result["width"] == 2048   # 2x scale
        assert result["height"] == 1536

    @pytest.mark.asyncio
    async def test_png_from_real_orbital(self, tmp_path):
        """실제 compute_ibo → HTML → PNG 전체 파이프라인."""
        from qcviz_mcp.backends.pyscf_backend import PySCFBackend
        backend = PySCFBackend()
        # water IBO → cube → HTML → PNG
        result = backend.compute_full_pipeline(
            "O 0 0 0; H 0 0.757 0.587; H 0 -0.757 0.587",
            basis="sto-3g"
        )
        html_path = result.get("html_path")
        if html_path and os.path.exists(html_path):
            from qcviz_mcp.renderers.png_exporter import html_to_png
            png_result = await html_to_png(html_path)
            assert png_result["success"]

    @pytest.mark.asyncio
    async def test_missing_playwright_graceful(self):
        """Playwright 미설치 시 graceful error."""
        # 이 테스트는 playwright가 설치된 환경에서는 의미 없음
        # mock으로 ImportError 시뮬레이션
        import unittest.mock as mock
        with mock.patch.dict('sys.modules', {'playwright': None, 'playwright.async_api': None}):
            # 모듈을 다시 import하면 ImportError 발생해야 함
            pass  # 구현 세부사항에 따라 조정

    def test_sync_wrapper(self, sample_html, tmp_path):
        """동기 래퍼 동작 확인."""
        from qcviz_mcp.renderers.png_exporter import html_to_png_sync
        result = html_to_png_sync(sample_html)
        assert result["success"]
```

## 제약 조건

1. **기존 95개 테스트를 절대 깨뜨리지 말 것.**
2. **전이금속 테스트는 `@pytest.mark.slow`로 마크** (def2-SVP SCF가 수십 초 소요)
3. **Playwright는 선택적 의존성** — 미설치 시 PNG 기능만 비활성화, 나머지 동작에 영향 없음
4. **UHF 코드는 기존 RHF 경로와 분리** — `isinstance(mf, UHF)` 체크로 분기, RHF 절대 불변
5. **ECP 폴백은 경고만** — 에러를 발생시키지 않고 결과에 warnings 리스트로 전달
6. **Fe(CO)5 좌표는 대략적** — 정확한 실험 구조가 아니므로 에너지 절대값 검증하지 않음, IBO 생성 여부만 확인
7. **PNG 테스트에서 네트워크 필요** — 3Dmol.js CDN 로드. 오프라인이면 skip.
8. **프로젝트 루트 확인 후 작업 시작**: `ls /mnt/d/20260305_양자화학시각화MCP서버구축/src/qcviz_mcp/`
